# PrepSmart - Placement Preparation Platform

## Overview

PrepSmart is a web-based Placement Preparation Platform designed to help students prepare for campus placements through aptitude practice, coding challenges, interview preparation, and AI-powered guidance.

## Features

### User Authentication

* User Registration
* User Login
* Secure Database Storage

### Aptitude Practice

* Company-wise aptitude questions
* Multiple-choice questions
* Score tracking
* Progress monitoring
* AI explanation for aptitude questions

### Coding Practice

* Company-wise coding questions
* Solution review
* Progress tracking
* AI coding hints

### Interview Preparation

* Company-specific interview questions
* Answer submission
* AI interview feedback
* Review previous answers

### Career Mentor Chat

* AI-powered career guidance
* Placement preparation advice
* Skill development suggestions
* Career roadmap support

### Progress Dashboard

* Track learning progress
* View quiz scores
* Monitor preparation status
